Visão Geral

O que é o NightStar

Problema resolvido

Preencher formulários de candidatura manualmente consome 20–40 minutos por vaga — e a maioria dos campos são sempre os mesmos. O NightStar automatiza isso em ~3 segundos de varredura + preenchimento, com revisão humana antes de enviar.

Funcionalidades

  • 3 camadas: mapa direto (40+ termos) → RAG vetorial → IA fallback
  • Suporte a Gupy, LinkedIn, Solides e formulários genéricos
  • Campos: texto, select, radio, checkbox, date, file upload
  • Detecção de captcha — pausa e aguarda resolução manual
  • Modo semi-auto: preenche tudo e para antes de enviar
  • Aprende mapeamentos por domínio e reutiliza em sessões futuras
  • Screenshots automáticos de cada etapa nos logs
  • Browser persistente com perfil Chrome — mantém login

Arquitetura

3 Camadas de Inteligência

01

Scanner — extrai campos do DOM

BeautifulSoup + heurísticas extraem labels, inputs, selects, radios, checkboxes de toda a página — incluindo iframes. Identifica plataforma (Gupy, LinkedIn…) e ativa adapter específico.

↓
02

Camada 1 — Mapa Direto (0ms)

Dicionário com 40+ termos comuns em PT/EN ("nome", "email", "linkedin", "cargo"…) resolve instantaneamente ~60% dos campos sem chamar IA.

↓
03

Camada 2 — Figurante mapeia (RAG)

Campo desconhecido vai para o Figurante com a lista de chaves do perfil. Retorna a chave correspondente + confiança (alta/média/baixa). Confiança baixa tenta modelo mais forte.

↓
04

Camada 3 — IA gera valor

Se não há correspondência no perfil, o Figurante gera o valor com base no contexto do campo. Resultados pendentes ficam para revisão humana.

↓
05

Semi-Auto — revisão antes de enviar

Preenche tudo, tira screenshot, pausa e exibe "Confirmar Envio" para o usuário revisar no browser antes de qualquer submissão.

Instalação

Como usar

1

Obtenha os arquivos

Opção A — Git:

git clone https://github.com/wenderson-werneck/nightstar

Opção B — Download: use o botão ⬇ Baixar Projeto Completo na seção Código-Fonte acima.

2

Pré-requisito: Figurante rodando

O NightStar usa o Figurante como cérebro. Certifique-se de que o app.py do Figurante está rodando na porta 5000.

3

Instale dependências

pip install playwright beautifulsoup4 ollama colorlog
playwright install chromium
4

Configure seu perfil

Edite data/perfil.json com seus dados. Configure documentos.curriculo_path para upload automático do PDF.

5

Execute via Figurante (recomendado)

No chat do Figurante, use o comando:

/signup https://url-da-vaga.com
