NightStar

Bot RPA que automatiza candidaturas a vagas de emprego. Varre formulários, mapeia campos ao perfil do candidato via RAG vetorial e preenche tudo automaticamente — parando antes do envio para revisão.

Ativo Python · Playwright ChromaDB · RAG Gupy · LinkedIn
Ver no GitHub Voltar

O problema

Candidatar-se a vagas é repetitivo ao extremo. Cada plataforma de RH (Gupy, LinkedIn, Workday) pede os mesmos dados — nome, email, experiência, formação — em formulários completamente diferentes, com fluxos diferentes e comportamentos diferentes. Horas por semana perdidas copiando e colando informações que já existem em outro lugar.

As 3 camadas de mapeamento

Decisão central de design: nunca chamar a IA para campos que podem ser resolvidos deterministicamente. Isso reduziu o tempo de varredura de ~40s para ~3s.

Campo: "Cidade de residência" │ [1] Mapa Direto → dict embutido com 40+ termos → 0ms │ (falhou) [2] RAG Vetorial do perfil.json │ ChromaDB local · embeddings offline · ~50ms │ (falhou: relevância < 0.60) [3] Campo Pendente │ Figurante chamado APENAS com autorização do usuário ✓ Ollama nunca chamado durante varredura

Fluxo completo

  • Você cola a URL da vaga no Figurante com /nightstar <URL>
  • Chrome abre com perfil persistente — login salvo de sessões anteriores
  • Você navega até o formulário; NightStar varre e mapeia campos
  • Relatório: campos mapeados (verde) e pendentes (amarelo) com inputs reais do formulário
  • Opção A: você preenche os pendentes manualmente nos cards
  • Opção B: clica em "🤖 Figurante" — IA resolve pendentes com base no seu perfil
  • Clica em "▶️ Preencher" — Playwright preenche tudo no browser em tempo real
  • NightStar para antes do envio — você revisa e confirma

Detalhe técnico: perfil persistente do Chrome

Em vez de abrir uma instância limpa do browser a cada execução, o NightStar usa launch_persistent_context do Playwright com um diretório local (nightstar_profile/). Isso preserva cookies, logins e histórico entre sessões — você faz login uma vez em cada plataforma; nas seguintes, os cookies já estão lá.
