Visão Geral

O que é o Figurante

Problema resolvido

LLMs na nuvem exigem internet, cobram por token e enviam seus dados para servidores externos. O Figurante elimina todas essas dependências: roda 100% local, com memória vetorial que persiste entre sessões.

Funcionalidades

  • Memória semântica via ChromaDB + embeddings ONNX offline
  • Dois modos: paz (1.5B, rápido) e ira (3B, preciso)
  • Upload de arquivos PDF, TXT, CSV, código no contexto
  • Streaming com cancelamento real via threading.Event
  • Múltiplas conversas com histórico isolado por chat_id
  • Regras editáveis inline pelo chat (/rules)
  • Modo Brain para ingestão em lote (/brain)
  • Integração nativa com o NightStar (/signup URL)

Arquitetura

Como funciona

01

Browser → Flask localhost:5000

Requisição HTTP/SSE. Cada conversa tem um chat_id único — histórico e tema (paz/ira) são isolados.

↓
02

ChromaDB — busca semântica

A pergunta vira embedding (all-MiniLM-L6-v2, 100% offline) e retorna as 4 memórias mais relevantes por similaridade coseno.

↓
03

Contexto enriquecido → Ollama

Memórias relevantes + regras do usuário + histórico da conversa são combinados no system prompt antes de ir ao modelo.

↓
04

Streaming SSE → Browser

Tokens fluem em tempo real. Cancelamento via threading.Event — sem esperar resposta completa.

↓
05

Destilação — ChromaDB aprende

Par pergunta/resposta é salvo de volta. A cada sessão o agente fica mais contextualizado, sem fine-tuning.

Instalação

Como usar

1

Obtenha os arquivos

Opção A — Git:

git clone https://github.com/wenderson-werneck/figurante

Opção B — Download: use o botão ⬇ Baixar Projeto Completo na seção Código-Fonte acima.

2

Instale o Ollama

Baixe em ollama.com e depois baixe os modelos:

ollama pull sam860/dolphin3-qwen2.5:1.5b
ollama pull sam860/dolphin3-qwen2.5:3b
3

Instale as dependências

pip install flask chromadb sentence-transformers ollama
4

Configure seu perfil

Copie perfil.example.json → data/perfil.json e preencha com seus dados reais.

5

Inicie o Figurante

python app.py

Acesse http://localhost:5000 no navegador.
