Visão Geral

O que é o Nexus AI

Problema resolvido

Buscar vagas manualmente em múltiplos sites gera resultados genéricos e não considera o fit real com seu perfil. O Nexus usa IA para expandir a busca e ranquear por semelhança semântica com suas habilidades.

Funcionalidades

  • Gemini expande 1 cargo em 15 títulos alternativos + 30 skills
  • Fallback offline quando Gemini não disponível (base hardcoded)
  • Adzuna API: busca em BR, US, UK, CA, AU, DE, FR e mais
  • Score em 2 eixos: afinidade com perfil + relevância da pesquisa
  • Penalização automática de vagas com inconsistência de integridade
  • Filtros de senioridade, faixa salarial, palavras negativas
  • Interface Streamlit com pesos ajustáveis em tempo real
  • Exportação dos resultados ranqueados

Arquitetura

Pipeline de Ranqueamento

01

Expansão via Gemini

Usuário digita um cargo. Gemini gera 15 variações de título + 30 skills relacionadas. Fallback: base de conhecimento offline por categoria (dados, dev, ML, RPA…).

↓
02

Fetch Adzuna — múltiplos países

Para cada título expandido, o NexusOrchestrator faz chamadas paginadas à API Adzuna. Deduplicação por ID para evitar vagas repetidas.

↓
03

Embedding + Score

SentenceTransformer (all-MiniLM-L6-v2) gera embeddings de cada vaga. Score final = (Peso Profile × sim_perfil) + (Peso Pesquisa × sim_query). Penalização por sênior/estágio se aplicável.

↓
04

Interface Streamlit

Resultados exibidos em tabela ordenada por score. Usuário ajusta pesos dos eixos em sliders. Acesso via ngrok no Colab.

Instalação

Como usar

1

Obtenha o notebook

Opção A — Git:

git clone https://github.com/wenderson-werneck/nexus-ai

Opção B — Download: use o botão ⬇ Baixar Projeto Completo na seção Código-Fonte acima. Depois faça upload do Nexus_AI.ipynb no Google Colab.

2

Configure as chaves (Colab Secrets)

No Colab, clique no ícone 🔑 na barra lateral → adicione os secrets:

GEMINI_KEY = sua_chave_google_ai_studio
ADZUNA_APP_ID = seu_app_id_adzuna
ADZUNA_APP_KEY = seu_app_key_adzuna
NGROK_TOKEN = seu_token_ngrok
3

Execute as células em ordem

Célula 1 (Setup) → Célula 2 (Engine) → Célula 3 (IA) → Célula 4 (Interface) → Célula 5 (Servidor). O link do dashboard aparece no output da Célula 5.

4

APIs gratuitas

Adzuna: cadastro gratuito em developer.adzuna.com — 1000 req/mês grátis.
Gemini: gratuito via Google AI Studio. ngrok: gratuito em ngrok.com.
