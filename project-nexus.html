Nexus AI

Agente inteligente de busca e ranqueamento de vagas. O Gemini expande um cargo em 15 variações e 30 skills, busca em 20+ países via API e ranqueia cada vaga pela similaridade semântica com o seu CV.

Ativo Gemini 1.5 Flash Adzuna API · 20+ países Streamlit · Google Colab
Ver no GitHub Voltar

O problema

Plataformas de emprego ordenam vagas por data de publicação ou palavra-chave exata. Isso significa que a melhor vaga para o seu perfil pode estar na página 5, enquanto na página 1 estão vagas irrelevantes publicadas hoje. O Nexus AI inverte essa lógica: ordena por compatibilidade real.

Pipeline completo

Você digita: "Machine Learning Engineer" │ [1] expandir_gemini() │ Prompt → Gemini 1.5 Flash → JSON │ Gera 15 variações de título + 30 skills │ Idioma detectado pelo país selecionado (pt/en/es) │ Fallback: PROFILE_DB embutido (sem internet) │ [2] NexusOrchestrator.fetch_adzuna() │ Para cada título: 2 páginas × 50 vagas │ Deduplicação por ID · Remove HTML das descrições │ 20+ países suportados │ [3] processar_ranking() │ Match_Profile: cos_sim(perfil_embedding, desc_embedding) │ Match_Pesquisa: título × skills com pesos configuráveis │ Penalização: -50 se empresa ausente ou link suspeito │ Match_Final: média ponderada (sliders da UI) │ ✓ Tabela ordenada por compatibilidade real

Score em dois eixos

Match_Profile — similaridade coseno entre o embedding do seu CV (profile.json) e a descrição de cada vaga. Mede o quanto a vaga realmente se alinha ao que você sabe e faz.

Match_Pesquisa — presença dos títulos gerados no título da vaga (peso configurável) + proporção de skills na descrição (peso configurável). Mede o quanto a vaga usa os termos da sua busca.

Você controla o equilíbrio entre os dois via sliders. O score final é a média ponderada configurada em tempo real — sem refazer a busca.

Fallback offline

Se o Gemini falhar (cota, conexão, chave inválida), o PROFILE_DB embutido assume automaticamente. É um dicionário com títulos e skills por área (dados, ML, dev) em três idiomas (pt, en, es). O agente nunca para por falta de API.
